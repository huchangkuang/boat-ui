# button-按钮
按钮用于开始的一个及时操作,标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。
## 组件注册
```vue
<script>
    import {Button} from "boat-ui"
    export default {
      components: {"b-button":Button}
    }
</script>
```
## 基本使用
使用方式：
<ClientOnly>
  <button-demo></button-demo>
</ClientOnly>
